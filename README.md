# ESP Fleet Management System

## Overview
IoT-based fleet management system using ESP32 + Web Dashboard.

## Features
- Real-time device tracking
- OTA updates
- MQTT communication
- Web dashboard visualization

## Folder Structure
- firmware/ → ESP-IDF project
- dashboard/ → Web app

## Tech Stack
- ESP-IDF (C)
- MQTT (EMQX)
- Node.js / React (or whatever you're using)

## Future Improvements
- GPS integration
- Geofencing
- Alerts system
